# **App Name**: EduStat Analytics

## Core Features:

- Academic Group & Student Enrollment: Enable the creation of academic groups and allow professors to upload student rosters from Excel (.xlsx) including validation (e.g., mandatory fields, unique student IDs). The system will persist student data in a database.
- Interactive Attendance Marking: Provide an intuitive interface for professors to start new class sessions, automatically logging the date. Professors can then manually mark each student as 'Presente' (1) or 'Ausente' (0), with a safeguard to ensure all students are marked before finalizing attendance for the class.
- Real-time Academic Performance Metrics: Automatically calculate and display essential academic statistics for each group and individual students, including individual attendance percentages, group average attendance, and standard deviation. The system also identifies students at risk based on configurable low attendance thresholds (e.g., < 80%).
- Advanced Numerical Analysis Visualization: Present various mathematical indicators derived from attendance data, explicitly showcasing the applied numerical methods. This includes calculations for Critical Approval Point (Bisection Method), Trend Projection (Lagrange Interpolation), and Cumulative Attendance Area (Trapezoid Rule).
- Multi-Group Comparative Dashboard: Offer comprehensive capabilities to select single or multiple groups for comparison. It displays combined statistics, group rankings based on average attendance, and comparative charts, allowing for deeper insights across different academic cohorts.
- AI-Powered Pedagogical Insights Tool: Implement a generative AI tool that provides clear, concise, and pedagogical explanations for the significance and real-world implications of each displayed numerical method and statistical indicator, aiding understanding for students and professors.

## Style Guidelines:

- Light color scheme to ensure optimal data readability and a clean, academic aesthetic.
- Primary color: Deep, authoritative blue (#243DA3) to signify trust and academic professionalism.
- Background color: Very light, subtle blue-grey (#F4F7FA) providing a clean canvas that visually supports the primary hue.
- Accent color: A vibrant and engaging aqua-green (#26D995) to highlight key actions, alerts, and positive data points, providing contrast without visual clutter.
- Unified body and headline font: 'Inter' (sans-serif) for its modern, objective, and neutral appearance, ensuring clarity and legibility across all data and textual content within the dashboard.
- Utilize a set of simple, crisp, and thematically relevant icons (e.g., checkmarks for attendance, charts for data, file uploads for excel, user groups) to enhance navigation and data comprehension.
- A modular and responsive dashboard layout with a clear hierarchy: group selection and key performance indicators prominently at the top, followed by dynamic and comparative chart areas for detailed analysis.
- Implement subtle, functional animations such as smooth transitions between data states, distinct loading indicators, and gentle hover effects, to provide intuitive feedback and enhance user interaction without distraction.
